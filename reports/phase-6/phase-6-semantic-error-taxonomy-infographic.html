V3 Backlog From Taxonomy

  • Normal hard negatives: เพิ่ม single failed login, routine health check, benign search ที่มีคำอย่าง sleep, select, union แต่ไม่มี attack pattern
  • SQLi contrast set: เพิ่มตัวอย่างที่ payload อยู่ใน /login หรือ username field เพื่อสอนว่า auth route ไม่ได้แปลว่า brute force เสมอไป
  • Traversal contrast set: เพิ่ม encoded traversal, Unix/Windows path, path probing และ paired normal static path ที่คล้ายกัน
  • Port scan contrast set: เพิ่ม nmap fingerprint, probed_ports, unique_ports, SYN scan detected พร้อม evidence สั้นและตรง
  • Prompt/schema wording check: ย้ำว่า brute force ต้องมี repeated failures หรือ volume signal, ส่วน evidence ต้องเป็น exact substring ที่เป็น signal จริง
  • Training render check: ตรวจ sample ที่ผ่าน tokenizer chat template ว่า assistant output เป็น raw JSON ล้วนและจบหลัง JSON ก่อนเริ่ม retrain v3
