Label Boundary Read

SQLi boundary

2/10 ถูกต้อง ส่วนที่เหลือกระจายไป brute force, traversal, normal และ port scan

ตีความ: training signal สำหรับ SQLi ยังไม่ชนะ bias ของ label อื่น แม้มี quote, tautology, UNION SELECT, SLEEP และ information_schema ใน supplement แล้ว

Traversal boundary

6/10 ถูกต้อง ดีขึ้นจาก canonical 4/10

ตีความ: runtime sampling ช่วยบางเคสของ traversal แต่ยังมี leakage ไป label อื่นอยู่

Port scan boundary

6/10 ถูกต้อง เท่ากับ canonical run

ตีความ: port scan ยังถูกดึงไป brute force 3 เคส และ normal 1 เคส จึงยังควรทำ boundary probe ต่อ

Decision

Keep as useful runtime probe

ค่า temp 0.3 ช่วยเล็กน้อยและไม่ทำให้ output contract พัง จึงควรเก็บไว้เป็น candidate สำหรับ probe เพิ่ม

Do not open fixed test yet

hard-contrast canary ยังไม่ถึงประมาณ 0.80+ และ SQLi ยังต่ำมาก จึงควรทำ targeted boundary tuning ก่อน Phase 7
